pip备忘录

pip介绍

  • 官方介绍：pip 是 Python 包管理工具，该工具提供了对Python 包的查找、下载、安装、卸载的功能。
  • 说白了就是可以用这个工具下载python第三方包，下载安装第三方包的方法很多，但使用pip来安装相比于其他方法可以省下不少麻烦

pip安装及配置

  • 检查是否安装

    • pip –version python2.x版本命令
    • pip3 –version python3.x 版本命令

  • 如果没安装则可以下载官方的安装脚本来进行安装

    • curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
    • sudo python get-pip.py

  • centos系统安装

    • yum update && yum clean 更新软件仓库
    • yum -y install epel-release 下载软件包扩展
    • yum -y install python-pip && pip install --upgrade pip

  • ubuntu系统安装

    • sudo apt upupdate && apt clean
    • sudo apt install python-pip && pip install --upgrade pip

  • 修改pip源

    • 默认的pip源为国外的，如果不修改的话，所有的第三方包都会从国外的镜像站进行下载，下载速度过慢不说，甚至连接超时下载失败。
    • 更换pip源，将镜像站修改为国内的就不会出现这种情况了
    #常用的pip源
# 豆瓣
https://pypi.doubanio.com/simple/
# 阿里云    
https://mirrors.aliyun.com/pypi/simple/
# 清华大学
https://pypi.tuna.tsinghua.edu.cn/simple/
https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple/

#临时使用某个镜像站
pip install some-package -i https://mirrors.aliyun.com/pypi/simple/

# 设置默认的镜像站
pip install -U pip 	# 更新
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/

pip 命令及参数

  • 这块呢，常用的就几个，安装/卸载/升级。

  • 安装包

    pip install SomePackage              # 最新版本
pip install SomePackage==1.0.4       # 指定版本
pip install 'SomePackage>=1.0.4'     # 最小版本
    • 比如安装Django则可以这样pip install Django==1.7

  • 升级包

    • 升级指定的包，通过使用==, >=, <=, >, < 来指定一个版本号。
    • pip install --upgrade SomePackage

  • 卸载包

    • pip uninstall SomePackage

  • 搜索包

    • pip search SomePackage

  • 列出以安装包

    • pip list
    • 查看可升级的包 pip list -o

  • 注意事项: python2与python3要同时安装或者使用pip的话则使用方法如下：

    • python2.x python2 -m pip install XXX
    • python3.x python3 -m pip install XXX
